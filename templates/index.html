Smart Electricity Billing Dashboard

A complete utility management platform built using Python, Flask, CSV-based data handling, analytics processing, and interactive visualisation dashboards.

Project Overview

This application simulates a real-world electricity billing system capable of generating bills using slab-based pricing logic, calculating taxes, penalties, discounts, overdue payments, and customer usage analytics in real time.

The system also includes customer management functionality, live dashboard analytics, graphical reports, and dynamic billing calculations integrated directly into the web interface.

⚡ Billing Engine

Slab-based electricity bill calculation with tax, discounts, penalties, and overdue tracking.

📊 Analytics Dashboard

Revenue insights, customer trends, payment monitoring, and usage analysis.

📈 Data Visualisation

Automated graph generation using Matplotlib for consumption and revenue reporting.

🛠 Customer Management

Add, edit, search, filter, and delete customer records dynamically through the UI.

🌐 Flask Web Interface

Multi-page responsive dashboard powered by Flask templating and modular backend logic.

💾 Structured Data Handling

CSV-based persistent storage designed with future SQL database scalability in mind.

🔄 Live Dashboard Updates

Analytics reports and charts update automatically whenever customer records are modified.

🔍 Search & Smart Filtering

Quickly locate customer records using integrated search, sorting, and filtering tools built for scalable utility data management.
